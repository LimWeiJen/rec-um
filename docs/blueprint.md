# **App Name**: REC Robotics Hub

## Core Features:

- Home Page Showcase: Display headline, a brief club overview, and quick links to other sections.
- About Us: Provide information on the founder, mission, and benefits of joining REC.
- Organization Structure: Show the current team hierarchy and list past directors of the club.
- Activity Feed: List past and upcoming activities, as well as the club's achievements.
- Robocon Focus: Outline what Robocon is, how the club prepares, team departments, and achievements.

## Style Guidelines:

- Primary color: Deep blue (#ffffff), reflecting engineering precision and trustworthiness.
- Background color: Light gray (#000b18), offering a clean, modern backdrop.
- Accent color: Electric blue (#e30606), used for highlighting interactive elements and important information, creating a visually engaging user experience.
- Body and headline font: 'Inter' for a clean and modern look that's easy to read.
- Use sleek, minimalist icons to represent sections and actions.
- Employ a grid-based layout to maintain order and balance across all pages.
- Use subtle animations for transitions and hover effects to add interactivity.
